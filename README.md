# 🛡️ AI Agent Simple Harness
> 一套轻量级、开箱即用的 AI Agent 安全运行框架，为 Claude Code / OpenClaw 等 Agent 提供边界约束、配置管理与日志追溯能力。

---

## ✨ 项目简介
本项目为 AI Agent 设计了一套基础 Harness（驾驭框架），解决了 Agent 常见的几个痛点：
- ❌ 越权操作、误删文件、访问隐私数据
- ❌ 配置混乱、参数不统一导致运行报错
- ❌ 无日志可追溯，出问题无法排查

它通过「安全边界+固定工作目录+统一配置+日志记录」四大模块，让 AI 既能高效干活，又始终在可控范围内运行。

---

## 📦 核心功能
- 🔒 **安全边界约束**：禁止高危操作、限制文件读写范围、拦截隐私访问
- 📂 **固定工作目录**：所有操作仅在指定目录内进行，避免误操作影响系统
- ⚙️ **统一环境配置**：集中管理邮箱、API、超时等参数，避免硬编码
- 📝 **全流程日志记录**：自动记录所有工具调用、命令执行和报错，方便回溯排查
- 🚀 **一键启动 Prompt**：预定义启动规则，每次运行自动加载全套约束
