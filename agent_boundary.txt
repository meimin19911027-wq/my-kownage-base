 定义 Agent 的安全边界规则，明确允许和禁止的操作。
  # Agent 安全边界规则
  # 最后更新: 2024-06-20

  ===== 允许的操作 =====
  1. 读取 GitHub/GitLab Wiki 内容（通过 API 或 Git 克隆）
  2. 编辑 Markdown 文件（需人工审核提交）
  3. 自动化同步 Wiki 到本地 JSON 文件
  4. 响应搜索请求（仅限公开内容）

  ===== 禁止的操作 =====
  1. 直接修改生产环境数据库
  2. 执行高风险 Shell 命令（如 `rm -rf`）
  3. 访问未授权的 API 或私有仓库
  4. 绕过权限检查操作 Wiki

  ===== 审计要求 =====
  - 所有编辑操作需记录到 `agent_logs/`
  - 触发敏感操作时通知管理员（如删除文件）
